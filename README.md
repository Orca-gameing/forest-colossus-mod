# Forest Colossus

A Minecraft Fabric mod that adds a massive tree guardian boss to jungle biomes.

## Features

- **Giant Tree Boss**: A 13-block tall colossus made of wood and stone
- **Tree Throwing**: Grabs actual trees from the terrain and hurls them at players
- **Ground Slam**: Devastating AoE attack that damages and launches nearby players
- **Glowing Eyes**: Eyes glow bright green at night
- **Jungle Spawning**: Rarely spawns in jungle biomes with a warning message
- **Boss Bar**: Full boss health bar during combat
- **Unique Drops**: Heart of the Colossus, Ancient Bark, and the Living Branch weapon

## Drops

| Item | Description |
|------|-------------|
| **Heart of the Colossus** | Epic rare drop, crafting material |
| **Ancient Bark** | Common drop (3-6), crafting material |
| **Living Branch** | Rare weapon with ground slam ability |
| **Logs** | 10-20 random wood logs |

## The Living Branch

A powerful weapon dropped by the colossus:
- Netherite-tier damage
- Right-click to perform a ground slam AoE attack
- Knockback and leaf particle effects

## Installation

1. Install [Fabric Loader](https://fabricmc.net/) for Minecraft 1.21.1
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Install [GeckoLib](https://modrinth.com/mod/geckolib)
4. Download the mod JAR and place it in your `mods` folder

## Requirements

- Minecraft 1.21.1
- Fabric Loader 0.16.0+
- Fabric API
- GeckoLib 4.7+
- Java 21+

## Commands

- `/summon forestcolossus:forest_colossus` - Spawn a colossus
- `/time set night` - Set to night to see glowing eyes
- `/give @p forestcolossus:living_branch` - Get the Living Branch weapon
- `/give @p forestcolossus:colossus_heart` - Get the Heart of the Colossus
- `/give @p forestcolossus:ancient_bark` - Get Ancient Bark
